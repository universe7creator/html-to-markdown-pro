Frequently Asked Questions

What HTML elements are supported?

We support all common HTML elements including headers (H1-H6), paragraphs, links, images, lists (ordered/unordered), tables, code blocks, blockquotes, horizontal rules, and inline formatting (bold, italic, strikethrough).

Is my data secure?

Absolutely! All conversion happens client-side in your browser. Your HTML/Markdown content is never sent to our servers or stored anywhere.

Can I convert Markdown back to HTML?

Yes! Our tool supports bidirectional conversion. You can convert HTML → Markdown and Markdown → HTML with equal accuracy.

Is there a file size limit?

The free demo supports content up to 50KB. The full version supports files up to 5MB, perfect for large documentation projects.
